## 현재 주요 기능

1. **개인별 알림 시스템**

   - 각 사용자의 주간 문제 해결 개수 자동 계산
   - 5개 미만일 경우 개인 DM으로 알림 발송
   - 개인 webhook URL 패턴: `{USERNAME}_MATTERMOST_URL`

2. **시간대별 알림**

   - 금요일 오전 9시: 주간 중간 체크
   - 일요일 오전 9시: 마감일 당일 알림
   - 일요일 저녁 9시: 마감 임박 긴급 알림

3. **전체 요약 알림**
   - 기본 채널로 전체 참가자 현황 요약 발송
   - 개인 알림과 별도로 운영
